# shevashen.com

Personal website for Sheva Shen, repositioned around AI platform leadership, enterprise AI adoption, agent systems, governance, and developer enablement.

## Stack

- Astro
- Markdown content
- Static deployment on Netlify

Legacy Gatsby source is preserved under `archive/legacy-gatsby-src` for reference while the public site runs from `src/`.

## Commands

```sh
npm install
npm run dev
npm run build
npm run serve
```

`npm run build` validates the production Astro output in `dist/`.
